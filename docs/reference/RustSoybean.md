# RustSoybean dataset

Soybean rust severity and field metadata.

## Usage

``` r
RustSoybean
```

## Format

A data frame with:

- detection:

  Detection score or date

- epidemia:

  Epidemic phase

- latitude:

  Latitude

- local:

  Location name

- longitude:

  Longitude

- planting:

  Planting date or stage

- severity:

  Disease severity

## Source

Del Ponte Lab internal data
