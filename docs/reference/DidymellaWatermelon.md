# DidymellaWatermelon dataset

Assessment of Didymella symptoms in watermelon plots.

## Usage

``` r
DidymellaWatermelon
```

## Format

A data frame with:

- EW_row:

  Row position (east–west)

- NS_col:

  Column position (north–south)

- dap:

  Days after planting

- severity:

  Disease severity

## Source

Del Ponte Lab internal data
