# BlastWheat dataset

Wheat blast dataset with severity and weather covariates.

## Usage

``` r
BlastWheat
```

## Format

A data frame with the following columns:

- heading:

  Date of heading

- inc_mean:

  Mean incidence

- index_mean:

  FHB index mean

- latitude:

  Latitude coordinate

- location:

  Experimental site name

- longitude:

  Longitude coordinate

- state:

  Brazilian state

- study:

  Study ID or code

- year:

  Crop year

- yld_mean:

  Mean yield

## Source

Del Ponte Lab internal data
