# FHBWheat dataset

Fusarium head blight quadrat assessments in wheat.

## Usage

``` r
FHBWheat
```

## Format

A data frame with:

- field:

  Field identifier

- i:

  Row position

- n:

  Column position

- quadrat:

  Quadrat ID

- season:

  Crop season

## Source

Del Ponte Lab internal data
