# FHBWheatBrazil dataset

Fusarium head blight (FHB) index in wheat in Brazil.

## Usage

``` r
FHBWheatBrazil
```

## Format

A data frame with the following columns:

- study:

  Study ID or code

- year:

  Crop year

- location:

  Experimental site name

- state:

  Brazilian state

- lat:

  Latitude coordinate

- lon:

  Longitude coordinate

- cultivar:

  Wheat cultivar

- planting_date:

  Planting date

- flowering:

  Flowering date

- index:

  FHB index

## Source

Del Ponte Lab internal data
