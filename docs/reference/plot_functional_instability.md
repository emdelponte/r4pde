# Plot method for functional instability results

Plot method for functional instability results

## Usage

``` r
plot_functional_instability(object, type = c("overall", "space", "time"), ...)
```

## Arguments

- object:

  Output from
  [`functional_instability()`](https://emdelponte.github.io/r4pde/reference/functional_instability.md).

- type:

  One of `"overall"`, `"space"`, or `"time"`.

- ...:

  Further arguments passed to methods.

## Value

A ggplot object.
