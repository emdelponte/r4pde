# WhiteMoldSoybean dataset

National dataset of white mold severity and yield.

## Usage

``` r
WhiteMoldSoybean
```

## Format

A data frame with:

- country:

  Country name

- elevation:

  Field elevation

- elevation_class:

  Elevation class

- harvest_year:

  Year of harvest

- inc:

  Incidence

- inc_check:

  Check plot incidence

- inc_class:

  Incidence class

- location:

  Location name

- region:

  Geographical region

- scl:

  Soybean canopy layer

- season:

  Crop season

- state:

  State name

- study:

  Study identifier

- treat:

  Treatment applied

- yld:

  Yield

- yld_check:

  Yield of untreated check

- yld_class:

  Yield class

## Source

Del Ponte Lab internal data
